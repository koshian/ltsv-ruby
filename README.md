# LTSV

LTSV parser and generator
Please see sample csv2ltsv and ltsview

## Installation

Add this line to your application's Gemfile:

    gem 'ltsv'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ltsv

## Usage

Please see sample csv2ltsv and ltsview
